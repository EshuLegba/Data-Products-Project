Introduction

This project makes it easier for a user to perform exploratory analysis on the mtcars dataset which was extracted from the 1974 Motor Trend US magazine and provides fuel consumption and 10 aspects of automobile design and performance for 32 automobiles (1973-74 models). A shiny app was developed to provide four basic exploratory analysis tools.

  • box plots displaying fuel mileage (mpg) as a function of each of the other variables
  • summary statistics of the data
  • table of a selected number of observations from the data
  • simple linear model of fuel mileage as a function of each of the other variables

Features of the Project

  • The project has four panels in the main panel window for each of the four exploratory data analysis tools. You can click on a panel to see a box plot, summary statistics, data table, or a simple linear model.

  • You can select the feature to be compared in the box plot and the linear model by using the Variable select box in the side panel.

  • You can select the number of rows to be displayed in the data table by using the Number of observations slider bar in the side panel.
